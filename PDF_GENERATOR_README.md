# PDF Documentation Generator

Generate comprehensive PDF documentation for the HC-Search-Models project.

## Overview

The `generate_documentation_pdf.py` script creates a professional PDF document containing:

- Project overview and statistics
- Architecture diagrams and explanations
- Model training & inference pipeline
- Core components breakdown
- Search pipeline detailed flowchart
- Technology stack details
- Model configuration parameters
- Key features and capabilities
- Language composition analysis
- Conclusions and future enhancements

## Installation

### Prerequisites

```bash
pip install reportlab pillow
```

### Full Requirements

```bash
pip install reportlab>=3.6.0 pillow>=9.0.0
```

## Usage

### Generate PDF

```bash
python generate_documentation_pdf.py
```

This will create a file named: `HC-Search-Models_Documentation.pdf`

### Output

The PDF will be generated in the current working directory with the following structure:

1. **Title Page** - Project name, repository info, and overview
2. **Table of Contents** - Complete document outline
3. **Project Overview** - Purpose, goals, and statistics
4. **Architecture Overview** - System layers and data flow
5. **Model Training & Inference** - Training and inference phases
6. **Core Components** - ML, data processing, and search layers
7. **Search Pipeline** - Detailed flowchart with all stages
8. **Technology Stack** - Complete tech stack with descriptions
9. **Model Configuration** - Parameters and entity configurations
10. **Key Features** - Feature list with benefits
11. **Language Composition** - Code breakdown by language
12. **Conclusion** - Summary and future enhancements

## Customization

You can modify the script to customize:

### Change Output Filename
```python
filename = "Custom_Name.pdf"
```

### Change Color Scheme
```python
title_style = ParagraphStyle(
    'CustomTitle',
    ...
    textColor=colors.HexColor('#YOUR_COLOR_HEX'),
    ...
)
```

### Adjust Page Size
```python
# Change from letter to A4
doc = SimpleDocTemplate(filename, pagesize=A4, ...)
```

### Modify Content
Edit the text strings in the script for any section to customize the documentation content.

## Integration with CI/CD

Add to your GitHub Actions workflow:

```yaml
- name: Generate Documentation PDF
  run: |
    pip install reportlab pillow
    python generate_documentation_pdf.py
    
- name: Upload PDF Artifact
  uses: actions/upload-artifact@v2
  with:
    name: documentation
    path: HC-Search-Models_Documentation.pdf
```

## Troubleshooting

### ImportError: No module named 'reportlab'
```bash
pip install reportlab
```

### PIL/Pillow Issues
```bash
pip install --upgrade pillow
```

### PDF Not Generated
- Check disk space
- Verify write permissions in current directory
- Ensure Python 3.7+
